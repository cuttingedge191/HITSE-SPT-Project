### 销售管理

#### 零售销售单状态

- POS创建、收款后直接`关闭(closed)`

#### 批发销售单状态

- 创建并提交后：`待审核(unchecked)`
  + 审核通过：`已审核并出库(checked)`
  + 审核拒绝：`审核未通过(failed)`
- 确认收款后：`已收款(paid)`
- 订单关闭：`已关闭(closed)`  **最终状态1**
- 申请退款/货：`待退款/货(refund)`
- 退款/货审核通过：`已退款/货(returned)` **最终状态2**
